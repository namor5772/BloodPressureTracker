# Blood Pressure Tracker

A simple Android app that lets you manually record your blood pressure with timestamps and view your history.

## Features

- **Record readings** — Enter systolic and diastolic values and save with an automatic timestamp
- **View history** — Scroll through all past readings, newest first, with formatted date/time
- **Local storage** — All data stored locally in an SQLite database using Room

## Tech Stack

- Kotlin
- Jetpack Compose with Material Design 3
- Room (SQLite persistence)
- Single Activity architecture
